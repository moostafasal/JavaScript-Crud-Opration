# Book Management App

A simple web application for managing books.

## Features

- Add new books with details like name, publish date, price, author name, and author email.
- Edit existing books.
- Delete books.
- View a table of all books.

## Technologies Used

- HTML/CSS/JavaScript
- Local Storage for data persistence
